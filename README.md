# Engineer Text Summary Generator

This project provides a summary generation service. To use the service, you need to set up your environment with the necessary API keys.

## Installation

1. Clone this repository.
2. Create a `.env` file in the root of the project.
3. Add the following line to your `.env` file:
   
   ```
   API_KEY=your_api_key_here
   ```
   
4. Make sure to replace `your_api_key_here` with your actual API key.

## Usage

After setting up the environment variable, you can run the application as per the instructions provided.
